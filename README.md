# Projeto de Ordenação de Cores em Java

Este é um projeto para a faculdade que ordena cores em RGB. O programa sorteia algumas cores e as ordena da menor para a maior soma e vice-versa.

## Como usar

Para usar este programa, siga estas etapas:

1. Clone este repositório para o seu computador.
2. Abra o projeto no seu IDE de escolha.
3. Execute o programa para ver as cores sorteadas e ordenadas.

## Contribuindo

Este é um projeto para a faculdade e não está aberto a contribuições externas. No entanto, sinta-se à vontade para clonar este repositório e experimentar com o código.
